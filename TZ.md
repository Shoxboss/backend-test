Необходимо:

1)  Разработать web сервис для получения информации о гео координатах и местоположении (страна, город) по IP-адресу (v4, v6).
    Запрос вида:
      http://<app_ip_port>/?ip=<ip>

    *  404 - нет данных по этому ip
